4. 計算実行¶

計算条件設定で、格子・格子属性を作成し、その他の計算条件を設定できたら次は計算を実行します。 「計算条件＞設定」から計算条件設定画面を表示し、「基本条件」を選択します。 「実行モード＞モード」で「計算実行」を選択し、「保存して閉じる」をクリックします。

「計算＞実行」をクリックすると計算が開始されます。
